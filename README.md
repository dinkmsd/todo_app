## Cài đặt
### 1. Clone project này (git clone https://github.com/dinkmsd/todo_app.git)
### 2. Cd vào file thư mục của project (cd todo_app)
### 3. Chạy lệnh `flutter run -d chrome` để chạy ứng dụng trên web brower Chrome
